# Database

Scripts SQL, diagramas y datos de prueba

## 01_schema.sql

Este script contiene la creación del esquema inicial de la base de datos del proyecto ClassAgenda.

Incluye:
- Creación de tablas principales
- Definición de claves primarias y foráneas
- Tipos de datos
- Relaciones entre entidades

## Cómo ejecutar el script 01_schema.sql

Este script crea automáticamente la base de datos y todas las tablas necesarias.

### Pasos:
1. Abre tu gestor de base de datos (MySQL Workbench, DBeaver o phpMyAdmin).
2. Abre el archivo `01_schema.sql`.
3. Ejecuta todo el script completo.
4. La base de datos se creará automáticamente y se generarán todas las tablas.
5. Comprueba que las tablas aparecen correctamente en el esquema.
